Command Description Example
ls (list) shows all of the major directories filed under a given file system 
ls /applications
will show the user all of the folders stored in the overall applications folder. The command is used for viewing files, folders, and directories.
cd (change directory) allows the user to change between file directories 
cd /arora/applications
switches to a directory called 'applications' in a directory called 'arora' in the folder the user is already in.
cd
without any other arguments simply switches to the home directory ~.
mv (move) allows the user to move a file to another folder or directory 
mv /s/a.txt /t
moves the file 'a.txt' in the directory /s to the directory /t.
man (manual) used to show the manual of the inputted command, which contains all relevant details regarding the usage of this command. 
man cd
shows the manual page, containing all options and other details, for the command 'cd'.
mkdir (make directory) allows the user to make a new directory 
mkdir testdirectory
creates a directory called 'testdirectory' in the directory the pointer is in.
rmdir (remove directory) removes an empty directory 
rmdir testdirectory
removes the empty directory 'testdirectory'.
touch allows the user to create an empty file 
touch test.txt
creates an empty .txt file called "test".
rm (remove) allows the user to delete files 

rm -rf /
removes all directories in the home directory and any files associated with it.

DO NOT TYPE THIS COMMAND IN THE COMMAND LINE!
locate searches the operating system for a file 
locate -i *red**house**city*
searches for a file with containing any of the three above words. 
-i
tells it to ignore capitalization; 
*[word]*
tells it to search for any and all files containing a word.
clear clears the terminal history 
clear
